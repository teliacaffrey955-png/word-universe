# 单词宇宙 · 单词探索器

一个游戏化的单词学习工具，把枯燥的背单词变成有趣的探索冒险。

## 🌟 功能特色

- 🎮 **游戏化学习**：每个单词都是一个游戏关卡
- 🧠 **深度理解**：不只是记意思，理解单词本质
- 📱 **全设备支持**：手机、电脑都能完美使用
- 🔄 **互动学习**：情景模拟、记忆游戏、画图创作
- 📊 **进度追踪**：可视化学习进度和成就

## 🚀 快速开始

### 在线使用
直接访问：[https://你的域名](https://你的域名)

### 本地运行
1. 下载所有文件到同一个文件夹
2. 用浏览器打开 `index.html`
3. 开始探索单词！

## 📁 文件说明

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 交互逻辑
- `words.json` - 单词数据库
- `manifest.json` - PWA配置

## ✨ 使用指南

### 探索单词
1. 点击左侧单词列表中的任意单词
2. 查看单词的核心隐喻和介绍
3. 点击卡片翻转查看详细规则
4. 在不同场景中练习使用
5. 使用记忆工具巩固记忆

### 互动功能
- **情景模拟**：选择不同场景，做出正确选择
- **画图记忆**：在画布上绘制记忆图像
- **进度追踪**：查看学习统计和成就
- **个性化设置**：切换主题、调整字体大小

### 快捷键
- `←` `→` 键：切换单词
- `ESC` 键：返回列表
- `Ctrl+F`：搜索单词
